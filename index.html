Reconciliation API

This website gathers various resources about the reconciliation API. It is maintained by the W3C Entity Reconciliation Community Group.

  • Specifications: defines the API;
  • Test bench: helps reconciliation service developers test their implementation interactively;
  • Census: lists software projects and protocols related to reconciliation;
  • Charter: the charter of our W3C Community Group.

This page can be edited at https://github.com/reconciliation-api/reconciliation-api.github.io.
